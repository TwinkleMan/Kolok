список команд
git status - статус проекта, состояние файлов
git add <имя файла> - добавление файла в очередь репозитория
git commit -m "<комментарии>" - подготовка обновления (коммита), комментарий обязательно должен содержать информацию о внесенных изменениях
git push - загрузка файлов на сервер
git pull - скачивание последних данных с сервера
